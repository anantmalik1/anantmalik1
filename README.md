Create a repo named exactly: anantmalik124

### **Step 3 — Snake Animation (GitHub Action)**

Create `.github/workflows/snake.yml` inside the **same repo** with this content:

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v3

      - name: Generate Snake
        uses: Platane/snk@v3
        with:
          github_user_name: anantmalik124
          outputs: |
            dist/github-snake.svg
            dist/github-snake-dark.svg?palette=github-dark

      - name: Push to output branch
        uses: crazy-max/ghaction-github-pages@v3
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Then go to **Actions** tab → click **"Generate Snake Animation"** → **Run workflow** manually the first time.

### **Step 4 — Replace usernames**
```
Find & Replace all instances of "anantmalik124"
with your actual GitHub username throughout the README.
```
